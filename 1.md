*	**import module**

		C에서의 include처럼 module을 불러올 수 있다.
		예를 들어 sqrt()를 쓰려면 import math로 math를 불어봐야된다.
		쓸떄는 math.sqrt()로 쓴다.

*	**from module import function**

		만약 함수만 쓰고싶다-> from module import function 이런 식으로 쓴다.
		math.sqrt()대신 sqrt()로 쓸 수 있다.
